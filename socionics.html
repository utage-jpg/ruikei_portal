ソシオニクスとは

Socionics — Overview

ソシオニクスは1970年代にリトアニアの社会学者アウシュラ・アウグスタ（A. Augusta）が提唱した心理類型論です。 ユングの機能論とアントニー・ケンピンスキーの「情報代謝」概念を融合させ、 人間の情報処理様式を16タイプに体系化しました。 MBTIと同じ記号（Te・Ti・Fe・Fi・Se・Si・Ne・Ni）を使いますが、理論体系・意味・タイプ間関係は別物です。
⚠️ MBTIとの記号の混用に注意
ソシオニクスとMBTIは同じアルファベット記号を使いますが、同一ではありません。 類型界隈では区別のために ソシオニクス = 🄢 Te（下線付き・サンセリフ）、 MBTI = Te（通常） と表記することが一般的です。このページでは「情報要素」はソシオニクスの意味で使います。

MBTIとソシオニクスの主な違い

① タイプコードの体系が異なる

ソシオニクスのタイプ名はMBTI式の「INTJ」ではなく「LII」「SLE」のような略称が基本です。 LIIは「Logical Intuitive Introvert」を意味し、主導機能（Ti）と創造機能（Ne）から名付けられます。 MBTIのINTJに対応するタイプは概ねLIIですが、完全な同一視はできません。

② 情報代謝（モデルA）が理論の中核

MBTIが4軸スコアや認知機能の優先順位を重視するのに対し、 ソシオニクスは8つのスロット（ポジション）に情報要素が割り当てられる 「モデルA」という構造を理論の中核に置きます。 スロットごとに「次元」「ブロック」「強度」が定義されています。

③ タイプ間関係論（インタータイプ関係）が発達

ソシオニクスの最大の特徴の一つがインタータイプ関係です。 14種の関係（双対関係・鏡像関係・活性化関係など）が数学的に定義されており、 2タイプの組み合わせが決まれば関係の種類が一意に決まります。 MBTIにはこれほど体系化された関係論はありません。

④ ロシア圏・東欧圏で発展した学問的背景

ソシオニクスはロシア・ウクライナ・リトアニアを中心に発展し、 英語圏ではWikisocionなどのコミュニティが理論を整理・英訳しています。 Viktor Gulenko（ウクライナ）は独自のモデルGやDCNH・小グループ理論などで理論を拡張しました。

情報要素

Information Elements（≠ MBTI Cognitive Functions）

情報要素はMBTIの認知機能と記号が同じですが、定義・使われ方が異なります。 MBTIの「Te（外向的思考）」がプロセスや動機を表すのに対し、 ソシオニクスの「Te（外向論理）」は情報代謝における「どの種類の情報を扱うか」を示します。 以下はソシオニクス文脈での定義です。
Te
外向論理 — 成果・効率
Extroverted Logic
現実における事実・実用性・効率に基づいて判断する。データ、方法論、実績を重んじる。
ソシオ：「これは機能するか・効率的か」という問い
Ti
内向論理 — 構造・定義
Introverted Logic
物事の体系・原理・内的整合性を重視。「理屈に合うか」「論理的に矛盾がないか」で判断する。
ソシオ：「この定義・構造は正確か」という問い
Fe
外向倫理 — 感情表出・場の空気
Extroverted Ethics
感情の表現・場の雰囲気を読む力。集団の空気を読み取り、感情を動かす。外に向かう感情エネルギー。
ソシオ：「場の感情状態をどう動かすか」という問い
Fi
内向倫理 — 関係性・内的価値
Introverted Ethics
人間関係の距離感・内的な好悪を感じ取る。「誰とどんな関係か」「相手の内心」が判断基準。
ソシオ：「この関係は親密か・距離があるか」という問い
Se
外向感覚 — 現実制圧・力
Extroverted Sensing
今この瞬間の物理的現実・力関係・状況支配をとらえる。行動力・存在感・制御を重視。
ソシオ：「誰がこの状況をコントロールしているか」という問い
Si
内向感覚 — 快適・身体感覚
Introverted Sensing
過去の感覚記憶・身体的快不快・心地よい環境を重視。安心できるリズムと安定を求める。
ソシオ：「この状況は快適か・身体的に心地よいか」という問い
Ne
外向直観 — 可能性・アイデア
Extroverted Intuition
対象から多様な可能性・アイデアを見出す。連想・創造力に富み、あらゆる方向に思考を拡げる。
ソシオ：「この状況にはどんな可能性が眠っているか」という問い
Ni
内向直観 — 時間・意味の核
Introverted Intuition
時間の流れ・因果・未来展開を洞察する。過去・現在・未来を一本の線で結び、本質を探る。
ソシオ：「この流れはどこへ向かっているか」という問い
価値機能 vs 非価値機能： 各タイプには「価値機能（自我・超イドブロック：1/2/5/6番）」と 「非価値機能（超自我・イドブロック：3/4/7/8番）」があります。 人は価値機能の情報に自然と引き寄せられ、同じ価値機能を持つ人と居心地よく交流できます。 非価値機能は意識的に使うと消耗しやすく、ストレスの元になります。

モデルA

Model A — Augusta's 8-Function Structure

モデルAはA. Augustaが構築したソシオニクスの中核理論です。 各タイプは8つのスロット（ポジション）に8種の情報要素が割り当てられ、 スロットごとに「ブロック」「次元」「強度」が定義されます。 スロット1〜8の配置がそのタイプの情報代謝の構造そのものです。

4つのブロック構造

自我ブロック
①
主導機能（Leading）
Dominant / Base
最も意識的・強力なスロット。自分のアイデンティティと一体化しており、自然体で使い続けられる。このスロットの情報は瞬時に認識・言語化できる。
自我ブロック
②
創造機能（Creative）
Auxiliary
主導機能を実行するための道具。主導より柔軟で状況に応じて使い方が変わる。目的達成のための手段として機能する。
超自我ブロック
③
規範機能（Role）
Tertiary
意識的に「こうあるべき」と思っているが、自発的には使いにくいスロット。過剰な期待をかけられると苦痛・罪悪感を感じやすい。
超自我ブロック
④
脆弱機能（Vulnerable）
Inferior / PoLR
最も弱く傷つきやすいスロット。批判を受けると深く傷つく。防衛的になりやすく、このスロットへの刺激を避けようとする。
超イドブロック
⑤
暗示機能（Suggestive）
Dual-seeking
最も強く補完を必要とするスロット。このスロットが得意な人と一緒にいると安心感・充足感を得られる。双対タイプが持つ主導機能と対応する。
超イドブロック
⑥
動員機能（Mobilizing）
Activating
補完されると活性化するスロット。⑤と同様に外から供給されることで能力が引き出される。双対タイプの創造機能と対応する。
イドブロック
⑦
無視機能（Ignoring）
Background
使えるが積極的には使わないスロット。意識的に活用するには労力がかかり、通常は無意識下で処理される。①と同じ情報要素の外/内逆バージョン。
イドブロック
⑧
証明機能（Demonstrative）
Demonstrative
表面上は使えるように見えるが、深く関わろうとはしないスロット。②と同じ要素の逆方向。外からはスキルがあるように見えるが本人は重要視していない。

次元（Dimensionality）

各スロットには「次元」という概念があり、1〜4次元でそのスロットの情報処理能力の深さを示します。

4次元（スロット①②） — 最高次元

経験・規範・状況・時間のすべてを統合して判断できる。最も自由で多様な使い方ができる。

3次元（スロット⑦⑧）

経験・規範・状況を使えるが、時間的変化への対応が弱い。

2次元（スロット③⑥）

経験と規範のみを使える。状況に応じた柔軟な対応が難しい。

1次元（スロット④⑤）

経験からのみ学べる最低次元。説明を受けても消化しにくく、失敗からしか学べない傾向がある。

モデルG

メンタルリング・バイタルリング

Gulenkoの「2つのリング」概念

モデルGではモデルAの8スロットを、メンタルリング（①②③④）とバイタルリング（⑤⑥⑦⑧）として再解釈します。

メンタルリング：意識的・言語化可能な処理。外部に表出しやすく、他者に説明できる機能群。
バイタルリング：無意識的・自動的な処理。本能・感覚・直感的な反応。言語化しにくく、表出しにくい機能群。

Gulenkoはこの2リングの循環によって情報処理の「意識↔無意識」の流れを説明します。

DCNHサブタイプ

GUlenkoが提唱するDCNHは、同じタイプ内でもどのスロットが活性化しているかでサブタイプを分類します。 D（ドミナント）・C（クリエイティブ）・N（ノーマライザー）・H（ハーモナイザー）の4種。 例えばLSE-D（ドミナントLSE）とLSE-H（ハーモナイザーLSE）は同タイプでも外見の行動パターンが異なります。

D — ドミナント

主導機能①が特に強調。自己主張が強く、リーダーシップを発揮。タイプの典型的な特徴が前面に出やすい。

C — クリエイティブ

創造機能②が特に強調。柔軟で適応力があり、問題解決・手段の工夫に長ける。

N — ノーマライザー

規範機能③が活性化。秩序・ルール・手順を重視し、調整役・管理役に向く。

H — ハーモナイザー

動員機能⑥が活性化。人間関係・雰囲気の維持に強く、和を大切にする。

認知スタイル・ロマンススタイル

認知スタイル（Cognitive Styles）

Gulenkoが提唱する4つの認知パターン。個々人が情報を処理し、問題を解決し、意思決定を行う際の特徴的な思考パターンから分類されます。

因果決定論的（CD）：ILE・LSI・SEE・EII
物事を原因と結果の連鎖で捉え、論理的・体系的に思考する。「なぜ」「その結果」を軸に構造的に問題を分解・分析。データや証拠に基づいた判断を好む。

弁証法的アルゴリズム的（DA）：SEI・EIE・ILI・LSE
対立や矛盾を統合しながら柔軟かつ予測的に思考を進める。条件分岐や動的バランスを活かし最適解を導く。複雑な問題・戦略分野に高い適性。

ホログラフィック・パノラマ的（HP）：LII・SLE・ESI・IEE
視点を自由に切り替えながら多角的に全体像を把握する柔軟なスタイル。複数視点を統合し物事を立体的に理解。危機や複雑な状況でも本質を見抜く。

ヴォーティカル・シナジェティクス的（VS）：ESE・IEI・LIE・SLI
試行錯誤と連鎖反応を通じて自己成長を遂げる動的で柔軟なスタイル。直感や実験的行動を通じ最適解を模索する。失敗を恐れず挑戦し続ける姿勢が強み。

ロマンススタイル（Romance Styles）

Gulenkoが提唱。恋愛やセクシャルな面における人と人との相互作用から分類された4パターン。侵略者↔犠牲者、保護者↔子供がそれぞれ相性◎の組み合わせ。

侵略者：SLE・LSI・SEE・ESI
他人への興味を明確に示し、ためらわずに行動する。恋愛では強さと優位性を重視し、関係の主導権を失っても弱さを見せない。

犠牲者：EIE・IEI・LIE・ILI
わずかに優位な強い相手に惹かれやすく、関係の変化に敏感で継続的な関心を求める。自他の関心に確信が持てず反応を見て探る。

保護者：ESE・SEI・LSE・SLI
穏やかで上品な交流を好み、相手の快適さと健康への配慮を重視する。攻撃的なセクシュアリティには冷めやすい。

子供：ILE・LII・EII・IEE
美的で非性的な魅力に惹かれやすく、楽しい会話や意外性で関心を引こうとする。自分のニーズを察する相手を高く評価する。

注意：モデルGはモデルAの「代替」ではなく「拡張・再解釈」です。 どちらが「正しい」かではなく、用途に応じて補完的に使われます。 Wikisocionや英語圏のコミュニティでは主にモデルAが参照されますが、 ロシア語圏ではGulenko理論も広く使われています。

ソシオニクス二分法

Socionics Dichotomies — Reinin Traits & Small Groups

ソシオニクスでは4つの基本二分法（外向/内向・直観/感覚・論理/倫理・合理/非合理）に加え、 Grigory Reinin が数学的に導出した15対の二分法（Reinin特性）があります。 Gulenkoはこれを小グループ理論として整理し、タイプの記述に活用しています。

エネルギー・態度系（外向/内向 軸）

外向 / 内向

外向：外部の人・状況・活動に意識が向き、外部から活力を得る。

内向：内面の思考・感覚・価値観に焦点を当て、内部から力を蓄える。

人付き合いのスタイルやエネルギーの充電方法に影響を与える。態度の控えめさと積極性の違いを表す。

周辺（賢明） / 中心（果敢）

周辺型：環境に溶け込み、調整や適応を重んじる。

中心型：自分の意思を強く押し出し、状況を動かそうとする。

守りか攻めかの志向の違いを表す。

α・δ（ILE,SEI,ESE,LII,IEE,SLI,LSE,EII）→ 周辺型　／　β・γ（SLE,IEI,EIE,LSI,SEE,ILI,LIE,ESI）→ 中心型

資源 / 利益

資源型：安定・安全・維持を優先する。

利益型：効率・成果・拡大を追求する。

守りか攻めかの志向の違いを表す。

臨機 / 先見

臨機型：状況に応じた即応を得意とする。

先見型：あらかじめ予測し、準備して動く。

「今対応する」か「先を読んで構える」かの違い。

情報処理系（直観/感覚 軸）

直観 / 感覚

直観：可能性や未来、抽象的な関連性に注意を向ける。

感覚：現在の状況や具体的な事実、実際的な体験に基づいて判断する。

情報をどう捉えるか、想像力と現実感覚のバランスを示す。

構成 / 情緒

構成型：体系的で秩序立った思考を重んじる。

情緒型：流動的で感情的なつながりを優先する。

堅実な計画性か、気分や雰囲気を大事にするかの差。

静的 / 動的

静型：安定・落ち着き・固定した枠組みを重んじる。

動型：変化・発展・流れを好む。

物事を「止まって見る」か「動かして見る」かの違い。

プロセス / 結果

プロセス型：途中の経過や進み方を重視する。

結果型：最終成果や結論に焦点を当てる。

「どう進んだか」か「何を得たか」に注目する違い。

判断・価値系（論理/倫理 軸）

論理 / 倫理

論理：客観的な法則や因果関係を重視し、合理性に基づいて判断する。

倫理：人間関係や価値観を重視し、調和や善悪を基準にする。

意思決定における「事実か価値か」の違いを表す。

主観 / 客観

主観型：自身の視点や解釈を基準に行動する。

客観型：外部の基準や状況に合わせて判断する。

「自分の物差しか、外界の物差しか」の違いを表す。

α・β（ILE,SEI,ESE,LII,SLE,IEI,EIE,LSI）→ 主観型　／　γ・δ（SEE,ILI,LIE,ESI,IEE,SLI,LSE,EII）→ 客観型

民主 / 貴族

民主型：仲間意識や平等な立場を重んじる。

貴族型：序列や区別を意識し、役割に基づく関係を取る。

横のつながりか縦のつながりかで人間関係を捉える。

肯定 / 否定

肯定型：物事を前向きにとらえ、可能性や価値を見出す。

否定型：問題点や矛盾を探し、修正や改善に向かう。

プラス思考か批判思考かという傾向を表す。

行動・コミュニケーション系（合理/非合理 軸）

合理 / 非合理

合理：判断機能（論理・倫理）を優先し、結論や安定を求める。

非合理：知覚機能（直観・感覚）を優先し、流れや変化を受け入れる。

計画的に進めるか、柔軟に適応するかの違いを示す。

戦術 / 戦略

戦術型：目の前の状況に柔軟に対応し、現場感覚に強い。

戦略型：長期的視点や全体像を重視し、未来を見据えて行動する。

短期適応か長期計画かのスタイルを示す。

質問 / 宣言

質問型：対話的で「相手から引き出す」スタイルを取る。

宣言型：一方的に「自分から伝える」スタイルを好む。

会話の主導権の取り方に差が出る。
