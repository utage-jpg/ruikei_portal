エニアグラムとは

Overview — Enneagram of Personality

エニアグラム（Enneagram of Personality）は、人間の性格を9つの根本的な動機・恐れ・欲求から記述する類型論です。 「エニア（9）＋グラム（図）」の名の通り、9点の円環図を基本構造とします。 グルジェフやオスカー・イチャーソが20世紀に体系化し、ドン・リチャード・リソ&ラス・ハドソン（The Enneagram Institute）らが現代的な形に整備しました。
⚠️ MBTIとの大きな違い
エニアグラムは「行動様式」ではなく「核となる動機・恐れ」を扱います。 同じタイプでも成長レベル（健全〜不健全）によって行動は大きく変わります。 外側から見える行動よりも、「なぜそう行動するのか」という内的動機こそが分類の核心です。

3つのセンター（知性の中枢）

9タイプは「本能（腸）センター」「感情（心）センター」「思考（頭）センター」の 3つのグループに分かれます。各センターは特有の中心感情を持ちます。

🔴 本能（腸）センター — タイプ 8・9・1

中心感情は「怒り・憤り」。身体感覚・本能的反応・力と自律を軸にします。 タイプ8は怒りを外に表出し、タイプ9は内に抑圧し、タイプ1は制御しようとします。

タイプ8 — 挑戦者 タイプ9 — 平和主義者 タイプ1 — 改革者

🟠 感情（心）センター — タイプ 2・3・4

中心感情は「悲しみ・恥」。アイデンティティ・イメージ・他者からの承認を軸にします。 タイプ2は他者に感情を向け、タイプ3はイメージを演出し、タイプ4は内省します。

タイプ2 — 援助者 タイプ3 — 達成者 タイプ4 — 個人主義者

🔵 思考（頭）センター — タイプ 5・6・7

中心感情は「恐れ・不安」。情報収集・安全・未来への備えを軸にします。 タイプ5は知識で不安に対処し、タイプ6はサポートを求め、タイプ7は回避します。

タイプ5 — 研究者 タイプ6 — 忠実者 タイプ7 — 熱狂者

エニアグラムの円環図と矢印（統合・分裂）

Enneagram Circle — 統合（→ 成長）/ 分裂（→ ストレス）

各タイプには統合（成長）の方向と分裂（ストレス）の方向があります。 健全に成長するとき、そのタイプは統合先のポジティブな特徴を取り込みます。 逆にストレス下では分裂先のネガティブな面が現れやすくなります。

矢印の読み方（例：タイプ1）

タイプ1の統合方向はタイプ7：健全な1は7の自発的・楽しさを受け入れ、 完璧主義から解放されます。
タイプ1の分裂方向はタイプ4：ストレス下では4の自己陶酔・メランコリーが現れます。

ウィング

Wing Theory

ウィング（Wing）とは、コアタイプの隣接タイプの影響を表す概念です。 タイプ4なら「4w3（ウィング3）」か「4w5（ウィング5）」のどちらかが強くなります。 ウィングはコアタイプを変えるのではなく、そのタイプの表れ方を修飾します。 両隣のウィングが均等な「イコール・ウィング」を持つ人も存在します。

全タイプのウィング一覧

ウィングの活用

なぜウィングを知るのか

同じタイプ4でも、4w3は社交的で表現力豊か（3の影響）、 4w5は知的で内省的（5の影響）と、外見上の行動パターンが大きく変わります。 タイプだけでは説明しにくい「同タイプなのに全然違う」という現象の多くは、 ウィングの差異で説明されます。

ウィングは一つとは限らず、状況によって両側の特徴が出ることもあります。 「どちらが優勢か」を見分けるには、どちらの隣タイプの特徴が自分に自然に感じられるかを内省することが有効です。

トライタイプ

Tritype® Theory — Katherine Fauvre

トライタイプはキャサリン・フォーヴルが体系化した概念です。 人は3つのセンター（腸・心・頭）それぞれに一つずつ支配的タイプを持ち、 その3タイプの組み合わせが「トライタイプ」を形成するとされます。 コアタイプが同じでもトライタイプが異なれば、行動パターンや悩みのパターンが大きく変わります。

例：コアタイプ4（心センター）＋腸センターで8が優勢＋頭センターで5が優勢 → トライタイプ 485（最初の数字がコアタイプ）

主要なトライタイプの例

腸センターが8の場合（8xx）

8-2-6（保護的リーダー） 8-3-7（達成主導型） 8-4-5（強烈な個）

心センターが4の場合（x4x）

1-4-5（完璧主義の内省者） 8-4-5（激しい個人主義者） 9-4-7（夢想的な自由人）

頭センターが5の場合（xx5）

1-4-5（分析的理想家） 5-3-9（静かな効率家） 6-2-5（慎重な援助者）

コアタイプとトライタイプの関係

「同じ4なのに全然違う」を解決する

4w3-845（コア4・腸8・頭5）は野心的で攻撃的な側面を持つ独自性の強い人物像を形成します。
4w5-469（コア4・腸9・頭6）は穏やかで不安傾向の強い、内省的で調和志向の人物像になります。

コアタイプだけでは説明できない細かな差異を、トライタイプが補完します。 コアタイプ→ウィング→トライタイプ→生得本能スタックと層を重ねることで より精緻な自己像が浮かびます。

注意：トライタイプはエニアグラムの派生理論であり、 公式化された研究機関による検証は限られています。 キャサリン・フォーヴルの研究（Tritype®）を出典とする非公式な拡張理論ですが、 類型界隈では広く使われている概念です。

生得本能（Instinctual Variants）

sp · sx · so — Instinctual Subtypes

生得本能（Instinctual Variants / 本能的サブタイプ）は、 人間が本能的に優先する3つの生存領域を示す概念です。 コアタイプと組み合わさることで、同タイプの中での行動差異を説明します。 優先度の高い順に並べた「スタック（積み重ね）」が重要で、 最下位の本能が最も盲点になりやすいとされます。
🛡️
sp
自己保存（Self-Preservation）
個人の安全・健康・物質的安定・快適さを優先する。食・住・財・体調に敏感で、日常の安全基盤を整えることに意識が向く。
安全健康資源節約計画性
⚡
sx
セクシュアル（Sexual / One-to-One）
強烈な1対1の繋がり・化学反応・魅力と吸引力を優先する。「この人と深く繋がりたい」という衝動が行動の根底にある。
強度魅力深い繋がり情熱刺激
🌐
so
ソーシャル（Social）
集団・コミュニティ・社会的役割・帰属意識を優先する。集団の中での立ち位置・貢献・社会的意味に意識が向く。
帰属役割貢献評判共同体

スタック（優先順位）

3つの本能は必ず優先順位を持ちます。上位から「支配的（dom）」「二次（sec）」「劣位（trd）」と呼ばれ、 6通りのスタックが存在します。スタックの最後に来る本能が最も使えない・意識外になりやすいとされます。

6通りのスタックと傾向

sp sx so

個人の安全と強烈な繋がりを重視。社会的意識が薄くなりがち。

sp so sx

安全基盤と社会参加を重視。1対1の強烈な繋がりには不慣れなことも。

sx sp so

強烈な繋がりと安全を重視。集団よりも特定の人との深みを求める。

sx so sp

情熱と社会参加を重視。自己管理・日常の安全基盤が盲点になりやすい。

so sp sx

社会参加と安全を重視。強烈な1対1の化学反応への関心が薄くなりがち。

so sx sp

社会参加と人間関係の深みを重視。身体的・物質的安全への配慮が薄くなりがち。

コアタイプ×生得本能の例（タイプ6）

6sp・6sx・6so — 同じ6でもこれだけ違う

6sp（自己保存6）：暖かみがあり、準備・備蓄・安全確保に積極的。反権威的な独立心が強く、「信頼できる人だけ」の小さなサークルを大切にする。
6sx（セクシュアル6）：強さ・権力・凄みで不安を打ち消そうとする。恐怖を感じると逆に攻撃的・反骨精神が出る（カウンターフォビック傾向）。
6so（ソーシャル6）：規則・権威・集団の方針に依拠することで安心感を得る。義務感が強く、体制への帰属で不安を安定させる。
